The map

In this part we will describe the core of our game, what objects compose the game, how they interact with each other and what they can do. This game is a 2D maze where the player evolves in a map composed by multiple frames in a 8x8 square

Our map will be contained in this array (not every spot will contain a part of the map) Each frame will have the same background but different layers of decoration. It will have a base size of (1280 x 960) They will have possibly 4 exits.
